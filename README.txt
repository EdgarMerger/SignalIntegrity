This is the PySI application.

To install this, simply unzip to a folder (called PySI).

To run the application, simply navigate to your working directory and type:

python [pathtoinstallation]/PySIApp/PySIApp.py

pathtoinstallation is the path to the file PySIApp.py in the root of the installation folder.

To open the help file (without running the application), open [pathtoinstallation]/PySIApp/Help/PySIHelp.html.LyXconv/PySIHelp.html.

Output Probes for Simulation

The Output Probe↓ defines the location, name and characteristics of an output Waveform↓ from the simulation.
They are added to the schematic by invoking either Add Output Probe↓ or by invoking Add Part↓ and selecting the Output Probe↓ from the Special category:
Note that the Output Probe↓ has configurable gain, offset, and delay that can be applied to the output waveform calculated. This is usually used to line the waveform up for better comparison with another waveform or to separate the waveforms vertically on the display.

Voltage Source

Property Name Type Keyword Units (Default) Value
default reference string VS?
reference string
file name file None
The voltage source is used in Simulation↑ applications to provide a voltage Waveform↓ specified by the file name to the system.
The Waveform↓ file can be viewed using Edit Properties↑ with the device selected and pressing the View button on the Part Properties dialog.

Errors and Exceptions

When problems crop up during calculation, PySIApp throws an exception and a message box is shown with the exception and the message the exception generates. As such, the exception type defines the main category of the exception and the message pertains to various flavors of that exception type.
The PySI exceptions are:
  • SystemDescription↓ - pertains to a general problem with the netlist used for calculation.
  • Numeric↓ - pertains to a general numerical error during calculation.
  • SParameterFile↓ - pertains to a problem with reading an s-parameter file.
  • WaveformFile↓ - pertains to a problem with reading a Waveform↓ file.
  • DeviceParser↓ - pertains to a problem parsing a device description in the netlist.
  • Simulator↓ - pertains to a problem specific to the Simulator application and calculation.
  • VirtualProbe↓ - pertains to a problem specific to the Virtual Probe application and calculation.

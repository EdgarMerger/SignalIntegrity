Export LaTeX (TikZ)

Dialog Main Schematic Dialog↑
Menu System File↑ ▸ Export LaTeX (TikZ)
Navigation Alt+F,L
Key Binding None
Toolbar None
Availability When the project is not empty
This causes the current schematic to be output in a graphical file format called TikZ which is interpreted in TeX and LaTeX based documents (see TeX Users Group (TUG).
While the TikZ file is directly importable into a LaTeX based document, it is difficult to edit. Fortunately a (old) free tool exists called TpX for creating graphics. The file format of a TpX file is actual TikZ code for the drawing with the TpX drawing information commented out. Thus, a TpX file is directly importable into a LaTeX based document and only the TikZ code is interpreted. PySIApp outputs the schematic this way, in TpX format so you can either directly import the schematic into a LaTeX based document, or if you install TpX, you can actually edit the schematic and save it as TikZ again.
Because of differences between graphics drawing commands, the TpX and TikZ will vary slightly from the actual schematic shown in PySIApp.
In the future, a .png or some other rasterized graphics version should be created, but believe it or not, this is more difficult. For now, you’re left with the screen capture methods built into the operating system to get this.

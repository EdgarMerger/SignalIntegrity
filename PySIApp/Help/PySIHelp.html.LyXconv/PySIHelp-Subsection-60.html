Measure Probe

Property Name Type Keyword Units (Default) Value
default reference string VM?
reference string
file name file None
The measure probe is used in Virtual Probing↑ applications to specify measurements actually made in the system. See Measure Probes↑.
The file specified is for a Waveform↓ file containing the actual measurement.
The Waveform↓ file can be viewed using Edit Properties↑ with the device selected and pressing the View button on the Part Properties dialog.

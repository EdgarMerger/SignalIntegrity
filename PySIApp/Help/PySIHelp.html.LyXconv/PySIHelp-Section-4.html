Introduction

If you want, go right to the Table of Contents.
PySIApp is an application for performing certain important applications and tasks encountered in signal integrity. These basic applications are:
  1. S-parameter Generation↓.
  2. Linear Simulation↓ of systems.
  3. Virtual Probing↓ systems.
  4. Deembedding↓s-parameters in systems.
The basic philosophy behind this tool can be summarized as follows:
  • A high percentage of tasks required by signal integrity engineers that are usually performed using expensive, closed, black box systems can be performed using this free, open-source tool.
  • While software tools are powerful and really help the signal integrity engineer, there are certain common mistakes made due to both quality of data input to the tools, and lack of basic understanding of what the tool is doing. We address these two issues by supplying many capabilities for understanding the quality of your input data, usually in the form of s-parameter files, and providing complete explanation of what the tool is doing all the way from theory (in the book that accompanies this software) and in the form of the software itself which can be examined and even modified.

Resample

Dialog S-parameter Viewer Dialog
Menu System Calculate ▸ Resample
Navigation Alt+C,R
Key Binding None
Toolbar None
Availability Always
When s-parameters are produced as a result of a S-parameter Generation↑ or Deembedding↑ application, they are sampled with the Calculation Properties↓ set at the time of the calculation. If they are changed after the calculation, the results can be resampled using this command.
Also, oftentimes s-parameters are being viewed from files and are therefore sampled however they are on the disk. Resampling them allows the user to see ahead of time how they will be resampled during calculation to look for bad behavior.

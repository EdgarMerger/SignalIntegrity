Open Help File

Dialog Main Schematic Dialog↑, S-parameter Viewer↑, and Simulator Dialog↑
Menu System Help↑ ▸ Open Help File
Navigation Alt+H,O
Key Binding None
Toolbar
Availability Always
This command opens this file that you are currently viewing from the application.
For the application to find the help system, it must be located at the relative directory to the file PySIApp.py at ./Help/PySIHelp.html.LyXconv/.
In this directory should be many help files with the root located in PySIHelp.html.

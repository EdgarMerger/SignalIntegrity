File

File tasks are a category in the Main Schematic Dialog↑. They are broken into two basic areas:
  • Dealing with the project file.
  • Exporting netlists and pictures of the drawing.
The File tasks are:
  • New Project↓
  • Open Project↓
  • Save Project↓
  • Save Project As…↓
  • Clear Schematic↓
  • Export Netlist↓
  • Export LaTeX (TikZ)↓

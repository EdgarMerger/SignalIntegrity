3 Introduction
4 ============
5 eLyXer converts a LyX source file to a HTML page. Full documentation in HTML
6 format can be found at docs/index.html, or on the web:
7  http://www.nongnu.org/elyxer/
8 
9 Installation
10 ============
11 Quick installation guide (for the impatient):
12 * download the latest version from http://www.nongnu.org/elyxer/,
13 * decompress the .zip or .tar.gz
14 * and install it using the provided script install.py as root:
15  # ./install.py
16  or on Windows:
17  > python install.py
18 
19 To install eLyXer first download a compressed version from
20  http://www.nongnu.org/elyxer/
21 You will also need a recent (> 2.4) version of Python on your target machine.
22 
23 For decompression: open a terminal window in the directory that contains the
24 downloaded file and just write at the command line prompt:
25  $ tar -xzf elyxer-[version].tar.gz
26 Or for the .zip version:
27  $ unzip elyxer-[version].zip
28 where [version] should be something like 0.44; the full name might be something
29 like elyxer-0.44.tar.gz.
30 
31 On Windows or Mac OS X you can unzip the file using the graphical tool of your
32 choice. In any case, a directory called elyxer-[version] should appear, where
33 the installer script install.py can be found (along with this README).
34 
35 The recommended installation procedure is to just run this script. On Linux
36 type as root:
37  # ./install.py
38 and similarly for Mac OS X, while for Windows open a console and type:
39  > python install.py
40 Note the you don't need to write the prompt, # or >; the console will print it
41 for you. Double-clicking on install.py should also work if your Python
42 installation is minimally sane. It will tell you as a result to which directory
43 eLyXer has been installed as a binary, which is be the typical result; in this
44 case eLyXer should be run as follows:
45  $ elyxer.py [input file] [output file]
46 or, on Windows:
47  > elyxer.py [input file] [output file]
48 You can test that it works with the --help option:
49  $ elyxer.py --help
50 or, on Windows:
51  > elyxer.py --help
52 Usage and options should then be shown.

54 LyX Integration
55 ===============
56 
57 To integrate eLyXer with LyX you just have to reconfigure LyX, selecting
58 Tools -> Reconfigure from within LyX. For rather old versions of LyX (< 1.6.2)
59 you may have to copy elyxer.py into the LyX directory, and reconfigure it.
60 Later versions will recognize eLyXer automatically, either if you install it
61 using distutils or as a binary.
62 
63 Usage
64 =====
65 eLyXer can be invoked from the command line as:
66  $ elyxer.py [source file] [destination file]
67 
68 If the source file is omitted then STDIN is used; likewise, if no destination
69 file is specified eLyXer will output to STDOUT. This allows its use in pipes
70 and other flexible configurations.
71 
72 Examples:
73  $ elyxer.py file.lyx file.html
74 converts file.lyx to file.html. Debug messages are shown.
75  $ cat file.lyx | elyxer.py > file.html
76 converts file.lyx to file.html, as before. This time debug messages are not
77 shown.
78  $ elyxer.py file.lyx | grep "<blockquote>" | wc
79 counts all blockquote paragraphs.
80  $ elyxer.py file.lyx | wget --no-check-certificate --spider -nv -F -i -
81 checks all external links in a document recursively. (Local links will appear
82 as unresolved, but they can be ignored.)

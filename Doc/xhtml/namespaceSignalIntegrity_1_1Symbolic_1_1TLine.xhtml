symbolic two-port transmission

symbolic two and four port single-ended transmission line based on rho and gamma

Parameters
Pinteger number of ports (2 or 4)
for two ports, returns SignalIntegrity.Symbolic.TLine.TLineTwoPort
for four ports, return SignalIntegrity.Symbolic.TLine.TLineFourPort
rhostring reflection coefficient
gammastring gamma
Returns
list of list of string s-parameter matrix containing LaTeX or ASCII strings for each element.
Note
strings can be any valid LaTeX
use SignalIntegrity.Symbolic.TLine.TLineRho if you want to enter the string representation of rho based on the characteristic impedance.
use SignalIntegrity.Symbolic.TLine.TLineGamma if you want to enter the string representation of gamma based on a time delay

Z0(optional) float reference impedance (defaults to 50 Ohms)
K(optional) integer number of sections (defaults to zero)
Note
Regarding number of sections, an approximate solution will be computed as a distributed line with the number of sections specified using the class TLineDifferentialRLGCApproximate, unless the number provided is zero (the default).
If zero is specified, then tests will be made for the appropriate analytic model to use according to the following table:
uncoupled balanced Model
True X TLineDifferentialRLGCUncoupled
False True TLineDifferentialRLGCBalanced
False False TLineDifferentialRLGCApproximate

If, because of imbalance and coupling, the approximate model must be used, the number of sections changed from K=0 specified to a value that will provided a very good numerical approximation.

The calculation is such that round-trip propagation time (twice the electrical length) of any one small section is no more than one percent of the fastest possible risetime.
